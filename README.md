## Compatibility

Name | Version
------- | -------
Ubuntu | >= 16.04
PHP | >= 5.5

## Installation

### Ubuntu

From PPA:

```bash
sudo add-apt-repository ppa:tarkhov/ppa
sudo apt update
sudo apt install composer-aliases
```

## Usage

### Commands

Alias | Command
------- | -------
`c` | `composer`
`cgr` | `composer global require`
`ci` | `composer install`
`cr` | `composer require`
`crd` | `composer require --dev`
`cu` | `composer update`
